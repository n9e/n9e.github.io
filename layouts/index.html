Out of the box

Nightingale has built-in alerting rules, metric descriptions, dashboards, etc. for various middleware and databases, allowing for out-of-box use. Additionally, it can integrate with popular collectors like Categraf, Telegraf, Datadog-agent, and various Exporters

Professional alerting

Nightingale supports alerting for both metrics and logs. A single set of rules can be applied to multiple data sources. Moreover, it supports features such as inhibit, mute, subscription, multiple notification mediums, and event relabeling.

Open community

Nightingale is hosted by the Open Source Development Committee of the China Computer Federation. With the active participation of thousands of community users, it ensures the healthy and sustainable development of the Nightingale open-source community.
